# FoodPlan AI clickable mobile prototype

Демонстрационный сайт сделан как интерактивный мобильный прототип внутри рамки смартфона. Это не статичная картинка: внутри экрана можно нажимать на элементы интерфейса и проходить основной сценарий продукта.

## Что должно быть видно на GitHub Pages

На десктопе сайт показывает landing-секцию и большой смартфон справа. Внутри смартфона открыт главный экран приложения: чат с LLM-помощником по питанию.

На мобильном браузере внешняя рамка убирается, и прототип занимает весь экран, как обычное мобильное приложение.

## Что кликается внутри смартфона

- Кнопка `☰` открывает боковую панель навигации.
- Пункты боковой панели переключают разделы: профиль, прогресс, планы питания, рецепты, список покупок, корзина, история и настройки.
- Быстрые действия вверху чата запускают сценарии: составить план, создать список покупок, показать рецепт.
- Кнопки внутри карточек добавляют новые сообщения и позволяют пройти путь: план питания → рецепт → список покупок → корзина.
- Поле ввода реагирует на запросы вроде `составь план`, `покажи рецепт`, `создай список покупок`, `собери корзину`.
- В карточке списка покупок можно отмечать чекбоксы.

## Основной пользовательский сценарий

1. Пользователь открывает сайт.
2. В центре или справа видит смартфон с мобильным приложением.
3. Нажимает `Составить план` внутри телефона.
4. Получает карточку плана питания.
5. Нажимает `Показать рецепт` или `Создать покупки`.
6. Получает рецепт, чеклист покупок и демо-корзину.
7. Через `☰` открывает боковую панель и переходит в профиль, прогресс или сохранённые разделы.

## Структура файлов

- `index.html` — HTML-разметка landing-страницы и экрана смартфона.
- `styles.css` — рамка смартфона, mobile-first интерфейс, чат, боковая панель и карточки.
- `app.js` — клики, открытие боковой панели, сценарии чата и интерактивные карточки.

## Как включить GitHub Pages

В настройках репозитория откройте **Settings → Pages** и выберите публикацию из ветки `main`, папка `/root`. После этого GitHub Pages будет отдавать `index.html` как демонстрационный сайт.
